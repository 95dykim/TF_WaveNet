# TF_WaveNet
An unofficial tensorflow/keras implementation of WaveNet: A Generative Model For Raw Audio (Oord et al, 2016)

https://arxiv.org/abs/1609.03499

# Requirements

tensorflow

tensorflow-datasets

librosa

pydub

numpy

pandas
